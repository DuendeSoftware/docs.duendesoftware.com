This section contains samples demonstrating SAML 2.0 integration with Duende IdentityServer.

### SAML 2.0 Basic

This sample demonstrates SP-initiated SAML 2.0 single sign-on using IdentityServer as the Identity Provider (IdP) and Sustainsys.Saml2 as the Service Provider (SP). It covers the full authentication lifecycle including login and single logout (SLO), orchestrated by .NET Aspire with automatic certificate generation.
