Duende User Management is a ready-to-use identity solution built on top of Duende IdentityServer and ASP.NET Core. It gives you a complete, extensible user store with a passwordless-first approach — OTP, TOTP, passkeys, external providers, user profiles, roles, groups, and multi-tenancy — without building identity infrastructure from scratch.
